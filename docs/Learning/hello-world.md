## Hello, World

プログラミング言語のチュートリアルなどで、最初に作るプログラムは、画面に"Hello, World"と出力するものであることが多い。この文書でもそこからはじめよう。

次のプログラムが、Hello, Worldプログラムである。これを実行すると、画面にHello, Worldが出力される。

```lisp
(princ "Hello, World!")
```

## Lispの基本構造

Lispのプログラムは全て丸括弧でくくられる。
